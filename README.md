# Portfolio Website

This project is a professional portfolio website for Jamie White. It is built to present selected case studies, experience, skills, and contact details in a cleaner employer-focused format.

## Stack

- React
- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- React Email
- Resend

## Local development

```bash
npm install
npm run dev
```

## Production checks

```bash
npm run lint
npm run build
```
